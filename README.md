<h1>Canteen Automation System</h1>

<p>The school needs their canteen sales and stock system advanced.</p>
<p>Languages for this project: PHP, HTML, JS, CSS</p>

<hr>

<h2>The school wants to track</h2>

<ul>
    <li>item stock</li>
    <li>sale numbers</li>
    <li>digital balance</li>
    <li>daily turnover reporting.</li>
</ul>


<h2>The school expects to have</h2>

<ul>
    <li>A fast sale user interface</li>
    <li>Item stock alert system</li>
    <li>User action history.</li>
</ul>
